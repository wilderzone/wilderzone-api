Welcome to the Wilderzone API

Endpoint Modifiers Description
/online/players ?simple

All players currently in-game.
/online/servers ?simple ?populated

All game servers currently online.
/history/players ?from=<timestamp> ?to=<timestamp> ?interval=<minutes>

A history of the total number of online players.
/news ?latest

News from the Wilderzone.

All data is provided as JSON.
